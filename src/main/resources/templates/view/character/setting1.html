〈모험자〉 제작

종족, 아키-메인 직업을 선택합니다.

〈모험자〉의 종족과 직업으로 【능력기본치】, 【최대HP】, 【초기 인과력】이 자동 계산됩니다.
단, 종족이 「휴먼」일 경우, 【능력기본치】에서 임의로 2개를 골라줍니다.
이후 【능력기본치】에 5점의 포인트를 배분할 수 있으며, 【능력기본치】는 7점을 넘어선 안됩니다.
【능력기본치】에서 3으로 나눈 값(나머지 버림)이 【능력치】가 됩니다.

종족 최대hp 0
메인직 초기인과치 0
str dex pow int p
종족 0 0 0 0 0
메인직 0 0 0 0 0
보너스포인트 0 0 0 0 0
능력기본치(합계) 0 0 0 0 0
능력치(능력기본치/3) 0 0 0 0 0

2. 자동 취득 특기의 확인(일부 선택)

〈모험자〉는 초기 선택에서 3개의 특기를 자동으로 취득하며, 이를 자동 취득 특기라 합니다.
※〈스워지버클러〉, 〈바드〉, 〈서모너〉, 〈인챈터〉의 경우「전투 특기」중 하나는 선택이 가능합니다.

자동습득기술
기술명 종별 sr 태그 특기사항
